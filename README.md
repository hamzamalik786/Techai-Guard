# TechuAI Guard (scaffold)

This directory contains the scaffold for the TechuAI Guard full-stack application.

Folders:
- `frontend/` — Next.js app (TypeScript + Tailwind)
- `backend/` — API, guardrails, AI provider adapters, Prisma schema
- `docker/` — docker-compose and infra
- `docs/` — architecture and API docs
- `tests/` — unit and integration tests

Use this scaffold as a starting point. Each service contains minimal starter files.
